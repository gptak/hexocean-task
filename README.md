## How to start

In the project directory, run:

### `npm install`

and then :

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Node and npm versions

Node.js v16.14.2.\
npm 8.5.0
